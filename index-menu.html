Pantallas de Aprendizaje

1

Origin (1985→1955)

Aprende los fundamentos de Git. Marty viaja accidentalmente a 1955 y debe aprender a navegar por el tiempo (y por Git) para regresar a casa.

Comandos Git
git init git add git commit git branch git checkout
🔒
2

Time Travel (2015)

Domina git checkout para viajar entre ramas. Marty y Doc viajan al futuro (2015) para salvar a la familia McFly de la ruina.

Comandos Git
git checkout [rama] git checkout -b navegación
🔒
3

Dystopia (1985A)

Aprende a arreglar historiales corruptos. Biff ha creado una línea temporal distópica y debes usar comandos avanzados para restaurar la realidad correcta.

Comandos Git
git reset --hard git revert git merge recuperación
🔒
4

Wild West (1885)

Domina git stash y comandos avanzados. Doc está atrapado en el Lejano Oeste y Marty debe rescatarlo mientras maneja múltiples cambios temporales.

Comandos Git
git stash git stash pop git branch -d gestión WIP
